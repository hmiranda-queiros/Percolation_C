Detailed Description

Simple definition of Cell_uf and Grid_uf.

Author
Hugo MIRANDA

This is a simple definition of Cell_uf and Grid_uf and the functions that allow to create and find and merge them. The API of union_find.h is defined as follows:

  • a function to create an union-find structure.
  • a function to find the subset to which an cells belongs.
  • a function to merge the subsets to which two cells belong.
