Detailed Description

Functions to read a text file in order to fill a grid and its corresponding structure.

Author
Hugo MIRANDA

This are functions to read a text file in order to fill a grid and its corresponding structure. The API of loader.h is defined as follows:

  • a function to read the shape of a grid from a text file.
  • a function to fill a grid from a text file.
