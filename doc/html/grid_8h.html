Detailed Description

Simple definition of Cell and Grid.

Author
Hugo MIRANDA

This is a simple definition of Cell and Grid and the functions that allow to create and print them. The API of grid.h is defined as follows:

  • a function to create a grid of empty Cells.
  • a function to print a grid.
  • a function to deallocate a grid.
