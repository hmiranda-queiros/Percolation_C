Detailed Description

Functions to generate a ppm image of a grid.

Author
Hugo MIRANDA

This are functions to generate a ppm image of a grid. The API of image_generator.h is defined as follows:

  • a function to generate a ppm image of a grid from a text file.
  • a function to generate a ppm image of a grid from the Monte-Carlo simulation.
